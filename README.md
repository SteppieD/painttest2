## 🚀 **Quick Start:**

### **For Development:**
```bash
npm install
npm run dev
# Visit: http://localhost:3001
```

### **For Docker:**
```bash
docker-compose build
docker-compose up -d
# Visit: http://localhost:3001
```

### **For Testing:**
1. **Customer Flow**: Enter access code → Create quotes → View dashboard
2. **Admin Flow**: Login at `/admin` → Manage customers → View analytics
