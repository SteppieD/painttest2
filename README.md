[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)

### **💻 Local Development:**
```bash
npm install
npm run dev
# Visit: http://localhost:3001
```

### **🐳 Docker:**
```bash
docker-compose build
docker-compose up -d
# Visit: http://localhost:3001
```
